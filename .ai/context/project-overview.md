# Project Overview

**easytocloud.github.io** is the public website for easytocloud, showcasing tools for AWS developers.

## Key Products

- **cloudX**: A modern successor to AWS Cloud9, enabling remote development with VSCode and EC2.
- **SSO Config Generator**: A tool to generate AWS CLI SSO profiles and directory structures.
- **Mac-letterhead**: A macOS utility for applying letterhead designs to PDFs and Markdown.

## Technology Stack

- **HTML5**: Semantic markup for structure.
- **CSS3**: Custom styling with CSS variables for theming (light/dark mode).
- **JavaScript**: Vanilla JS for interactivity (mobile menu, theme toggle, scroll animations).
- **GitHub Pages**: Hosting platform.

## Design Principles

- **Consistency**: Uniform header, footer, and navigation across all pages.
- **Accessibility**: High contrast colors, semantic HTML, and ARIA attributes.
- **Responsiveness**: Mobile-friendly design with a hamburger menu for smaller screens.
- **Dark Mode**: First-class support for dark mode, respecting system preferences.
